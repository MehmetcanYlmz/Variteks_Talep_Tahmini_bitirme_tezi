# Variteks_Talep_Tahmini_bitirme_tezi
Variteks firmasının ürün bazlı satış verilerini kullanarak 3 aylık talep tahmini projesini yaptım. Bu proje Yıldız Teknik Üniversitesi Endüstri Mühendisliği bitirme çalışması olarak hazırlanmıştır ve juriye sunulmuştur. Amacımız firmanın kullanabileceği bir tahminleme çalışması yapmaktı.
